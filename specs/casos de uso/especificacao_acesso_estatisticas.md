# Especificação de Caso de Uso: Acesso às Estatísticas (Homepage) [RF011]

**Ator Principal:** Visitante (Anônimo) ou Usuário Logado
**Prioridade:** Importante
**Complexidade:** Média

## Descrição
O sistema deverá exibir as estatísticas das partidas realizadas durante o JES em sua homepage pública.

## Fluxo Principal
1. O usuário acessa a homepage pública do Portal JES.
2. O sistema exibe as estatísticas e resultados das partidas recentes.
3. O usuário utiliza filtros para buscar partidas específicas por modalidade ou atlética.
4. O sistema apresenta os dados filtrados em tempo real (com atraso máximo de 1 dia conforme [RNF012]).
