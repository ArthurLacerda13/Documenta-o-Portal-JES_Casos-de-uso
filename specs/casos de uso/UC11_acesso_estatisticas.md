# Caso de Uso: UC11 - Acesso às Estatísticas (Homepage)

**Objetivo:** Exibir estatísticas públicas das partidas.
**Requisitos:** [RF011]
**Atores:** Visitante
**Condição de entrada:** O ator acessa a URL pública do portal.

## Fluxo Principal
1. O sistema carrega dados de partidas recentes.
2. O ator aplica filtros de busca.
3. O sistema exibe os resultados.
