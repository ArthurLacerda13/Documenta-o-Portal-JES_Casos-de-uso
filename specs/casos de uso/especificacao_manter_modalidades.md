# Especificação de Caso de Uso: Manter Modalidades [RF005]

**Ator Principal:** Moderador
**Prioridade:** Essencial
**Complexidade:** Média

## Descrição
O sistema deve permitir ao Moderador realizar o cadastro das modalidades esportivas que compõem o campeonato.

## Dados Necessários
- Nome da modalidade
- Categoria (Masculino, Feminino, Misto)
- Tipo de modalidade (Coletiva, Individual)
- Limite mínimo de atletas por equipe
- Limite máximo de atletas por equipe
- Vagas por atlética

## Fluxo Principal
1. O Moderador acessa a gestão de Modalidades.
2. O Moderador define as configurações da modalidade (limites de atletas e vagas).
3. O sistema salva a modalidade para uso posterior nos cadastros de equipes e partidas.
